Introduction:
    This is a small Bank Simulation in which I try to learn some skills.

Features of the Alpha version:
    - output in German -> trying to translate it when I have time.
    - 2 main account models Admin and User.
    - each User has a purse and an account.
    - you can send money between your purse and your account and between other Users.
    - "Databank" for Users and Admins. 
    - basic security features like password hashing.

Have fun!
